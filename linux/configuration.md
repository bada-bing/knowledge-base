   - Easier Prompt Definitions. Particularlly, dynamic/multiline prompts
   - Smart & Fast Autocompletions (case-insensitive, smart defaults)
   - Path Expansion (e.g., `cd /v/w [TAB]` => `cd /var/www`)

   - Keybindings (google how to do it)
